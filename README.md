# CS50 SQL - Cyberchase Problem Set

Solution for CS50 SQL database problem set. Analysis of Cyberchase TV series data.

## Description

Database contains 140 episodes including season, title, topic, air date, and production code.

## Tasks Completed

- **1.sql** - All episodes from season 1 (26 rows)
- **2.sql** - First episode of each season (14 rows)
- **3.sql** - Production code for "Hackerized!" (CYB091)
- **4.sql** - Episodes without topic (26 rows)
- **5.sql** - Episode count in season 1 (26)
- **6.sql** - First and last air dates (2 dates)
- **7.sql** - Top 6 in reverse alphabetical order (6 rows)
- **8.sql** - Episodes from 2018 (5)
- **9.sql** - Episodes about fractions (6)
- **10.sql** - All episodes with title, topic, air_date (140 rows)
- **11.sql** - 10 most recent episodes (10 rows)
- **12.sql** - Episodes from 2011-2013 (3)
- **13.sql** - Top 5 most popular topics (5 rows)

## Technologies

- SQLite3
- SQL (SELECT, WHERE, ORDER BY, LIMIT, COUNT, LIKE)
